# Novelpia Offline
[taeseong14](https://github.com/taeseong14)님의 [N-down](https://github.com/taeseong14/N-down)을 기반으로 만든 안드로이드 노벨피아 Downloader/Reader 앱 입니다.  

---

# 기능
## 소설 다운로드
노벨피아 계정으로 로그인 후 소설을 다운로드하여 저장할 수 있습니다.

## 오프라인 읽기
다운로드한 소설을 오프라인에서 앱을 통해 읽을 수 있습니다.
